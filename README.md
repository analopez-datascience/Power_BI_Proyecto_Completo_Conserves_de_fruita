# 📊 Fruit Canning Company Analytics | Power BI Business Intelligence Project

## 📌 Project Overview

This project is a Business Intelligence case study developed in **Power BI**, based on a fruit preserves company.

The objective is to analyze operational and sales data in order to answer key business questions related to:

- Distribution network optimization.
- Product performance.
- Sales evolution over time.

The project involves data import, cleaning, transformation, data modeling, and the creation of visual reports to support business decision-making.

---

## 🎯 Business Context

The company needs insights to improve its logistics and sales strategy. The analysis focuses on three main business questions:

1. Where should the distribution center be located?
2. Which product has the highest number of units sold?
3. In which month did the highest sales occur?

## 🔄 Data Preparation & Transformation

Data preparation was performed using **Power Query (M Language)**.

### Main steps:

- Data import from multiple tables.
- Data cleaning and standardization.
- Handling missing values.
- Column formatting and type correction.
- Data consistency validation.

---

## 🔗 Data Modeling

A relational data model was created to ensure correct analysis across all datasets.

### Relationships defined:

- `Capital → Capital_Prov`
- `Código Producto → Producto`

### Key modeling tasks:

- Table relationships configuration.
- Ensuring no disconnected tables.
- Data integrity validation.
- Model optimization for reporting performance.

## 🧠 Analytical Questions & Visual Solutions

### 📍 Question 1: Where should the distribution center be located?

- A **geographical map visualization** was created using Power BI.
- Bubble sizes represent different quantities or metrics.
- Location accuracy was validated.
- Geographic formatting applied for clarity.

---

### 📦 Question 2: Which product has the highest number of units sold?

- A **bar/column chart** was created to compare product performance.
- Sales distribution across products was analyzed.
- Visualization maintained consistency with previous formatting.

---

### 📈 Question 3: In which month did the highest sales occur?

- A **time series chart** was used to analyze sales evolution over time.
- Monthly aggregation of units sold.
- Identification of peak sales periods.
